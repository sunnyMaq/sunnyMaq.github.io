MacBook 轻于时代，先于时代。

  • 观看影片《设计》
探索 MacBook
  • 观看影片《设计》

对于 MacBook，我们给自己设定了一个几乎不可能实现的目标：在有史以来最为轻盈纤薄的 Mac 笔记本电脑上，打造全尺寸的使用体验。这就要求每个元素都必须重新构想，不仅令其更为纤薄轻巧，还要更加出色。最终我们带来的，不仅是一部新款的笔记本电脑，更是一种对笔记本电脑的前瞻性思考。现在，有了第六代 Intel 处理器、提升的图形处理性能、高速闪存和最长可达 10 小时的电池使用时间*，MacBook 的强大更进一步。

现有银色、金色、深空灰色，以及新增的玫瑰金色可供选择。

Retina 显示屏，
满屏创新，及至前沿。

当你打开 MacBook，绚丽的 12 英寸 Retina 显示屏，再配上无边式玻璃，让一切都看起来格外光彩夺目。照片犹如跃然屏幕之上，细腻生动，纤毫毕现。超过 300 万个像素，令每个文字都显得清晰锐利。所有这些，都呈现于这块 Mac 电脑有史以来最为纤薄的 Retina 显示屏之上，并在超乎想象的简约设计中雕琢出震撼的视觉体验。

进一步了解设计

全尺寸键盘，
置身更纤薄空间。

我们相信，要让笔记本电脑有出色的使用体验，一个舒适的全尺寸键盘必不可少。但是，要想把一个全尺寸键盘置入精致优雅的 MacBook 中，就必须对整个键盘的构造和工艺进行重新构想。我们重新设计了每个按键和它的基础结构，不仅让整个键盘变得更薄，同时还带来更为舒适、精确，更为得心应手的输入体验。

进一步了解设计

Force Touch， 让触控板更进一步。

MacBook 带来了一种截然不同的触控体验。在 Force Touch 触控板的设计中，无论你点按触控板表面的哪个位置，都能得到灵敏且效果一致的点击响应。此外在触控板下方，还有力度感应器可检测你的点按力度。现在你可以通过用力长按来启用一系列新功能。比如，只需在触控板上点击并按住不放，即可快速查看词语定义或预览文件。你还可以体验触觉反馈，触控板会发出触觉振动，让你在屏幕上看到的内容还可以通过触觉感知。所有这些先进功能，还可与深受 Mac 用户喜爱的 Multi-Touch 手势来配合使用。轻轻松松间，你与 Mac 之间的沟通，迈入新的境界。

进一步了解设计

为无线世界，
全面装备就绪。

MacBook 为轻松融入这个越来越无线化的世界而设计。有了支持新一代 Wi-Fi 和蓝牙技术的 Apple 软件，你在笔记本电脑上的许多操作，现在几乎都可以无线完成。你可以轻松接入网络、传输文件、整理照片、听听音乐等，无需被线缆束缚。

进一步了解无线连接

USB-C，集多才多能于一身。

在为 MacBook 设计充电端口时，我们希望把它打造成一个融合了各种先进技术和多重功能的端口。USB-C 端口正如我们期望的那样，将你需要的不同端口功能集于一身。除了充电，这款出众的端口还可进行 USB 3 快速数据传输以及视频输出。它采用双面可用的设计，大小仅仅是 USB 3 端口的三分之一，让你能够轻松灵活地连接喜爱的设备。

进一步了解设计

种种所需，能一一为你连接。

我们对 MacBook 如何连接外设和电源进行了深入再深入的思考，最终决定采用设计紧凑、灵活多用的 USB-C。只需这一个端口，你便可连接充电器、HDMI、DisplayPort、VGA 显示器、外置硬盘等 USB 设备，以及你的 iPhone 或 iPad。看，有时候用用减法，也能成就更多。

静静的震撼

为追求安静、高能效的表现，MacBook 彻彻底底重新设计。我们采用了运行功率只有 5W 的第六代 Intel Core M 处理器，再对 OS X 进行整体优化，令其能效进一步提升。由于处理器和 OS X 如此高效节能，整个系统的发热量很小，因此不需要风扇为电脑散热。所以在 MacBook 运行期间，你不会听到它有什么声音。不仅如此，承载处理器的主板也经过精心设计，将所有你期待的 Mac 功能，密集整合在尽可能小巧的空间内。

进一步了解设计

提速显而易见

MacBook 采用速度最高可达 1.3GHz 的第六代 Intel Core m3、m5 和 m7 处理器，以及速度提升最高可达 25% 的图形处理器。另外，它还配备速度更快的 1866MHz 内存。因此，比起之前的 MacBook，不仅运行速度更快，响应也更灵敏。

满足一天所需的电池续航，
现更持久。

由于 MacBook 的机身格外纤薄，如果使用传统矩形电池，将无法满足使用一天的电力需求。我们专门开发了自己的电池技术，以充分利用每一丝可用空间。由此，一种等高线型阶梯式电池应运而生。它独到的设计，能与 MacBook 内部的纤薄空间巧妙贴合。而现在，因为有了更高效的处理器，以及进一步改进的电池化学技术，电池使用时间也延长了一小时。

进一步了解设计

OS X

有了 OS X 这一操作系统，你便能通过 Mac 来一展所长。OS X El Capitan 既能让你轻松成就非同寻常之事，也能让你愉快处理一切日常之事。不仅如此，它还能与你的各种 app 和 iOS 设备默契协作。

进一步了解 OS X

内置 App

每台 Mac 都配备了众多激发创意和提升效率的 app。同时，还拥有一系列出色的 app 来处理各种日常事务，包括浏览网页，发送邮件和信息，以及整理你的日历。它甚至还有一款 app，可以帮你查找新的 app。所以，你的 Mac 不仅功能齐备，而且全副武装。
